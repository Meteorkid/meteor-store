# 微信收款码配置指南

## 如何生成微信收款码

### 方式一：微信收款助手（推荐）

1. 打开微信
2. 搜索 **"微信收款助手"** 小程序
3. 点击 **"收款码"**
4. 选择 **"保存收款码"**
5. 保存到手机相册

### 方式二：微信支付

1. 打开微信
2. 点击 **"+"** → **"收付款"**
3. 点击 **"二维码收款"**
4. 点击 **"保存收款码"**

### 方式三：微信商家助手（如果有）

1. 打开 **"微信商家助手"** App
2. 登录你的账号
3. 进入 **"收款码"** 页面
4. 下载高清收款码

---

## 如何上传收款码

### 步骤 1：获取收款码图片

从手机相册中找到收款码图片，发送到电脑。

### 步骤 2：重命名文件

将图片重命名为：`wechat-qr.png`

### 步骤 3：替换文件

将图片复制到以下路径：

```
/Users/meteor/github/meteor-store/public/wechat-qr.png
```

### 步骤 4：重新部署

```bash
cd /Users/meteor/github/meteor-store
git add .
git commit -m "feat: 更新微信收款码"
git push
```

Vercel 会自动部署更新。

## 注意事项

### ⚠️ 重要

1. **收款码要清晰** - 确保二维码可以扫描
2. **金额设置** - 微信收款码不支持固定金额，用户可以扫码后自行输入
3. **订单确认** - 用户支付后需要发送订单号到邮箱确认

### 💡 建议

1. **使用高清收款码** - 避免模糊
2. **定期检查** - 确保收款码有效
3. **记录订单** - 手动记录支付信息

## 测试收款码

### 方式一：使用测试收款码

微信没有官方的测试收款码，但你可以：

1. 用自己的微信生成收款码
2. 用另一个微信扫码测试
3. 支付小额（如 0.01 元）测试

### 方式二：使用沙箱环境

微信没有官方沙箱，建议：

1. 先用支付宝测试
2. 微信收款码作为备用

## 常见问题

### Q: 收款码有有效期吗？

A: 微信收款码长期有效，但如果长时间不用可能会失效，建议定期检查。

### Q: 可以设置固定金额吗？

A: 微信收款码不支持固定金额，用户扫码后需要手动输入金额。

### Q: 收款码有手续费吗？

A: 个人收款码没有手续费，但提现到银行卡可能有手续费。

### Q: 如何查看收款记录？

A: 在微信 → 我 → 服务 → 钱包 → 账单 中查看。
