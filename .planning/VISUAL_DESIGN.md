# Meteor Store - 视觉设计方案

## 设计理念

**"一眼吸引，瞬间转化"**

打造一个具有视觉冲击力的现代化软件销售网站，通过动态效果和精美设计吸引用户注意力，提升转化率。

---

## 核心视觉元素

### 1. 粒子背景动画

**效果描述**: 首页 Hero 区域添加动态粒子背景，粒子随鼠标移动产生交互效果。

**技术实现**:
- 使用 Canvas 或 Three.js
- 粒子颜色: 紫色渐变 (#667eea → #764ba2)
- 交互: 鼠标移动时粒子跟随
- 性能: 使用 requestAnimationFrame 优化

**参考**: Vercel、Stripe 首页效果

---

### 2. 渐变流光效果

**效果描述**: 按钮、卡片添加流光渐变效果，鼠标悬停时光效流动。

**技术实现**:
- CSS 渐变动画
- 鼠标位置追踪
- 光效跟随鼠标
- 平滑过渡动画

**参考**: Linear、Raycast 按钮效果

---

### 3. 3D 卡片倾斜

**效果描述**: 产品卡片支持 3D 倾斜效果，鼠标移动时卡片产生透视变化。

**技术实现**:
- CSS 3D transforms
- 鼠标位置计算
- 透视效果
- 光影变化

**参考**: 苹果官网产品卡片效果

---

### 4. 滚动视差

**效果描述**: 页面滚动时，不同层级元素以不同速度移动，产生深度感。

**技术实现**:
- CSS transform: translateZ
- 滚动事件监听
- 性能优化: 使用 Intersection Observer

**参考**: Apple、Tesla 官网效果

---

### 5. 数字滚动动画

**效果描述**: 统计数字从 0 滚动到目标值，增强数据展示效果。

**技术实现**:
- 数字递增动画
- 缓动函数
- 滚动触发

**参考**: 各大 SaaS 官网统计区域

## 动画效果详细设计

### 1. 页面加载动画

```css
/* 页面加载时的加载动画 */
.loading-animation {
  animation: fadeInUp 0.6s ease-out;
}

@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

### 2. 卡片悬停效果

```css
/* 卡片悬停效果 */
.card {
  transition: all 0.3s ease;
  transform-style: preserve-3d;
}

.card:hover {
  transform: translateY(-10px) rotateX(5deg);
  box-shadow: 0 20px 40px rgba(139, 92, 246, 0.3);
}
```

### 3. 按钮流光效果

```css
/* 按钮流光效果 */
.button-glow {
  position: relative;
  overflow: hidden;
}

.button-glow::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: linear-gradient(
    90deg,
    transparent,
    rgba(255, 255, 255, 0.3),
    transparent
  );
  transition: left 0.5s ease;
}

.button-glow:hover::before {
  left: 100%;
}
```

### 4. 文字渐变动画

```css
/* 文字渐变动画 */
.gradient-text {
  background: linear-gradient(135deg, #667eea, #764ba2, #f093fb);
  background-size: 200% 200%;
  -webkit-background-clip: text;
  background-clip: text;
  -webkit-text-fill-color: transparent;
  animation: gradientShift 3s ease infinite;
}

@keyframes gradientShift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
```

### 5. 滚动触发动画

```css
/* 滚动触发淡入 */
.scroll-fade-in {
  opacity: 0;
  transform: translateY(30px);
  transition: all 0.6s ease;
}

.scroll-fade-in.visible {
  opacity: 1;
  transform: translateY(0);
}
```

## 技术栈选择

### 动画库

- **Framer Motion**: React 动画库，支持页面过渡、手势动画
- **GSAP**: 高性能动画库，支持复杂动画
- **CSS Animations**: 简单动画效果

### 性能优化

- **Intersection Observer**: 滚动触发优化
- **requestAnimationFrame**: 动画性能优化
- **will-change**: CSS 性能优化
- **debounce**: 事件节流

### 响应式设计

- **Mobile First**: 移动端优先设计
- **Breakpoints**: 响应式断点
- **Fluid Typography**: 流式字体
